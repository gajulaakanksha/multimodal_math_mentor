# Algebra — Formulas, Identities & Solution Templates

## Algebraic Identities
- (a + b)² = a² + 2ab + b²
- (a - b)² = a² - 2ab + b²
- (a + b)³ = a³ + 3a²b + 3ab² + b³
- (a - b)³ = a³ - 3a²b + 3ab² - b³
- a² - b² = (a + b)(a - b)
- a³ + b³ = (a + b)(a² - ab + b²)
- a³ - b³ = (a - b)(a² + ab + b²)

## Quadratic Equations
- Standard form: ax² + bx + c = 0
- Quadratic formula: x = (-b ± √(b² - 4ac)) / 2a
- Discriminant D = b² - 4ac
  - D > 0: two distinct real roots
  - D = 0: one repeated real root (x = -b/2a)
  - D < 0: two complex conjugate roots
- Sum of roots: α + β = -b/a
- Product of roots: αβ = c/a

## Sequences & Series
- Arithmetic Progression (AP):
  - nth term: aₙ = a + (n-1)d
  - Sum of n terms: Sₙ = n/2 × (2a + (n-1)d) = n/2 × (a + l)
- Geometric Progression (GP):
  - nth term: aₙ = ar^(n-1)
  - Sum of n terms: Sₙ = a(rⁿ - 1)/(r - 1), r ≠ 1
  - Sum of infinite GP (|r| < 1): S∞ = a/(1 - r)
- Sum of first n natural numbers: n(n+1)/2
- Sum of squares: n(n+1)(2n+1)/6
- Sum of cubes: [n(n+1)/2]²

## Inequalities
- If a > b, then a + c > b + c for any c
- If a > b and c > 0, then ac > bc
- If a > b and c < 0, then ac < bc (inequality flips!)
- AM ≥ GM: (a+b)/2 ≥ √(ab) for non-negative a, b
- Cauchy-Schwarz: (a₁b₁ + a₂b₂)² ≤ (a₁² + a₂²)(b₁² + b₂²)

## Logarithms
- logₐ(mn) = logₐm + logₐn
- logₐ(m/n) = logₐm - logₐn
- logₐ(mⁿ) = n·logₐm
- logₐb = logₓb / logₓa (change of base)
- logₐa = 1, logₐ1 = 0
- Natural log: ln(e) = 1, ln(1) = 0

## Absolute Value
- |x| = x if x ≥ 0; |x| = -x if x < 0
- |x| < a ⟺ -a < x < a
- |x| > a ⟺ x < -a or x > a
- |x + y| ≤ |x| + |y| (Triangle inequality)
